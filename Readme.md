# 🎯 주제
  -  Spring Boot와 JSP를 활용하여 구축되는 전자제품 쇼핑몰 
  -  관리자 시스템, 고객 관리, 상품 관리, 주문 처리, 배송 관리 등의 핵심 기능을 제공하는  웹 애플리케이션

# 주요기능
- 회원가입 및 로그인
- 상품 목록 조회(카테고리별 조회)
- 장바구니 추가/삭제/수량 변경
- 고객 주문 및 결제
- 관리자 상품 등록/수정/삭제 기능
- 주문 내역 확인 및 주문 상품 관리
- 고객 주소지(배송지) 관리
